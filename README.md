## ✨ 功能特色

### 核心功能
- 🎯 **即時投票系統** - 所有投票狀態即時同步，溝通零延遲。
- 🔗 **網址直接分享** - 支援通過房間連結直接加入，隨點隨到，無需手動搜尋房間。
- 📱 **QR Code 快速接入** - 內建 QR Code 生成器，手機掃描即可秒速加入會議。
- 🎨 **自訂房間名稱** - 重複名稱自動編號（Room - 2, Room - 3...），組織更清晰。
- 🧹 **過期自動清理** - 閒置或過期房間自動偵測刪除，確保系統輕量、高效率。

### 房主專屬權限 (Host Controls)
- 👁️ **結果揭示 (Reveal)** - 房主掌控開票時機，確保估點過程公正不被干預。
- 🔄 **開啟新回合 (New Round)** - 一鍵重置所有成員投票，快速開始下一項任務估點。
- 🛡️ **成員管理 (Kick)** - 房主擁有最高權限，可將誤入或干擾的人員移出房間。
- 🚫 **智慧封鎖 (Ban)** - 被踢除者短時間內無法再次潛入原房間，維持會議秩序。

### UI/UX 優化
- ⏳ **平滑載入視覺** - 整合全域 Loading 動畫，操作反饋優化。
- 🌟 **金色 consensus** - 投票揭示後，系統會自動標記並凸顯最具共識的數值。
- 🌑 **專業深色模式** - 護眼且高品質的視覺介面。
- 📱 **完美響應式** - 手機、平板、桌面端一致的極致體驗。

## 🛠️ 技術棧

### 前端
- **React 18** + **React Router** - 現代化導航與 UI 框架
- **Zustand** - 高效狀態管理
- **Tailwind CSS 4.0** - 預設深色系優化樣式方案
- **Socket.IO Client** - 極速即時通訊

### 後端
- **Node.js** + **Express** - 輕量化後端支撐
- **Socket.IO** - 雙向 WebSocket 通訊
- **UUID** - 高安全性的唯一識別碼生成

## 📖 使用說明

### 1. 建立房間
進入首頁輸入您的名字與房間名稱，點擊「Create Room」即可成為房主。

### 2. 邀請成員
您可以透過兩種方式邀請夥伴：
*   **複製連結**：點擊上方導航列的「連結圖示」即可複製。
*   **QRCode**：點擊「QR Code 圖示」開啟視窗供現場掃描。

### 3. 主持會議
*   成員陸續加入並完成投票。
*   點擊 **Reveal Votes** 顯示結果。
*   討論結束後，點擊 **New Round** 清空票數開始下一輪。
*   若有人員誤入，點擊成員卡片右上角的 **X** 即可將其移出。

## 🎨 特色功能展示

### 金色凸顯最多票
當 Reveal 投票結果時，得票最多的選項會自動以金色漸層凸顯，幫助團隊快速達成共識。

**凸顯規則：**
- ✅ 明確多數：凸顯金色並震動
- ⚠️ 票數平手：不凸顯（避免誤導）
- 📊 全部相同：不凸顯
- 👤 單一投票：不凸顯

### 房間名稱智能編號
重複的房間名稱會自動加上編號，讓團隊能輕鬆創建多個相似用途的房間：
- `Daily Standup` → `Daily Standup - 2` → `Daily Standup - 3`
